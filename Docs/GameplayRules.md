# Gameplay Rules

## Current Ritual Rules

- Players stay seated.
- Players sit automatically when the ritual begins.
- A ready check starts the ritual.
- There is one real cursed book.
- The book moves player to player.
- The phrase starts with 1 word.
- Every player says the same current phrase.
- After a full table rotation, add 1 word.
- The active player must complete the full visible phrase before the hourglass expires.
- Whisper is the primary voice recognition system.
- Windows speech recognition is fallback only.

## Out Of Current Scope

- Notebook.
- Cards.
- Lore delivery.
- Demon reactions.
- Unity Dictation.
- Azure voice services.
